## Questões desenvolvidas

### 1. Controle de vendas com classificação

O algoritmo permite registrar valores de vendas até que o usuário digite `0` para encerrar.

Cada venda é classificada como:

* Venda baixa: até R$100;
* Venda média: de R$101 até R$500;
* Venda alta: acima de R$500.

Ao final, o programa apresenta:

* Total vendido;
* Quantidade de vendas;
* Quantidade de vendas em cada categoria.

---

### 2. Sistema de notas com situação do aluno

O algoritmo registra notas de alunos até que o usuário digite `-1` para encerrar.

Cada aluno é classificado como:

* Aprovado: nota maior ou igual a 7;
* Recuperação: nota entre 5 e 6,9;
* Reprovado: nota menor que 5.

Ao final, o programa apresenta:

* Média da turma;
* Quantidade de alunos aprovados;
* Quantidade de alunos em recuperação;
* Quantidade de alunos reprovados.

---

### 3. Controle de estoque com alerta

O algoritmo permite informar uma quantidade inicial em estoque e registrar retiradas de produtos.

O programa atualiza o estoque após cada retirada e exibe alertas quando necessário:

* Estoque baixo: quando o estoque fica menor que 10;
* Estoque esgotado: quando o estoque chega a 0.

Ao final, o programa apresenta:

* Estoque restante;
* Total retirado;
* Quantidade de vendas realizadas.

---

### 4. Pesquisa de temperaturas com alerta

O algoritmo registra temperaturas até que o usuário digite `999` para encerrar.

Cada temperatura é classificada como:

* Frio: abaixo de 15°C;
* Agradável: entre 15°C e 30°C;
* Quente: acima de 30°C.

Ao final, o programa apresenta:

* Média das temperaturas;
* Quantidade de dias frios;
* Quantidade de dias agradáveis;
* Quantidade de dias quentes.

---

### 5. Sistema de login com tentativas limitadas

O algoritmo simula um sistema simples de login.

As credenciais corretas são:

* Usuário: `admin`;
* Senha: `1234`.

O usuário possui até 3 tentativas para acertar os dados.

O programa pode exibir:

* Acesso permitido;
* Dados incorretos;
* Acesso bloqueado.
